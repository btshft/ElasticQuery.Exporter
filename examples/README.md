# Examples

This folder contains quickstart examples:
* `1-multiple-queries` - run exporter with multiple queries (on demand and scheduled). 

All examples are used a local Elasticsearch as data provider. You can start it by running command `docker-compose up` inside `_shared` directory. To configure sample data look at [official guide](https://www.elastic.co/guide/en/kibana/7.8/getting-started.html#get-data-in).
